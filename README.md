🌸 Iris Flower Classification using Decision Tree

📌 Project Overview

This project is a Machine Learning classification model built using the famous Iris dataset. The model predicts the species of an iris flower based on features like sepal length, sepal width, petal length, and petal width.

The project uses the Decision Tree Classifier algorithm to classify flowers into different categories with high accuracy.

🧠 Model Used
Decision Tree Classifier

The Decision Tree model works by creating a tree-like structure of decisions based on feature values to make predictions.

🎯 Objective

To classify iris flowers into:

Setosa
Versicolor
Virginica

based on flower measurements.

⚙️ Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
🚀 Workflow
Load Iris dataset
Explore and preprocess data
Split dataset into training and testing sets
Train Decision Tree model
Evaluate model accuracy
Predict flower species
📊 Features Used
Sepal Length
Sepal Width
Petal Length
Petal Width
📈 Model Performance

The Decision Tree model achieves high classification accuracy on the testing dataset and successfully predicts iris flower species.

▶️ How to Run
pip install -r requirements.txt
python iris_model.py
🌟 Conclusion

This project demonstrates the implementation of a supervised machine learning classification algorithm using Decision Trees and provides a beginner-friendly introduction to classification problems in machine learning.
